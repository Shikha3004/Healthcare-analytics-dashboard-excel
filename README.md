This project is an interactive healthcare analytics dashboard created using Advanced Microsoft Excel.
The dashboard provides insights into patient data, hospital performance, and revenue trends to support data-driven decision-making.

Key Features

Dynamic KPIs for:

Average Length of Stay

Average Billing Amount

Total Revenue

Revenue analysis by insurance provider.

Admission type distribution (Emergency, Urgent, Elective).

Doctor performance metrics.

Medical condition frequency analysis.

Tools Used

Microsoft Excel (Advanced)

PivotTables

Pivot Charts

Slicers

Data Validation

Insights Generated

Identified high-revenue insurance providers.

Analyzed patient admission patterns.

Evaluated doctor performance trends.

Compared revenue growth over time.

Purpose

This project demonstrates practical skills in:

Data analysis

Dashboard design

KPI reporting

Business intelligence using Excel
